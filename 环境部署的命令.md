# 环境部署命令（完整可复现）

以下命令在 **Linux + CUDA 12.8 + H800 GPU** 环境下验证，可用于快速搭建微信人格 LoRA 训练环境。

## 1. 创建 Conda 环境（Python 3.11）
```bash
conda create -n wechat python=3.11 -y
conda activate wechat
```

## 2. 安装 PyTorch（CUDA 12.8 版本）
```bash
pip install torch==2.7.1 torchvision==0.22.1 torchaudio==2.7.1 --index-url https://download.pytorch.org/whl/cu128
```

## 3. 安装训练依赖（固定版本，避免冲突）
```bash
pip install transformers==4.56.0 peft==0.12.0 datasets accelerate bitsandbytes numpy==1.26.4
```

## 4. 验证安装
```bash
python -c "import torch, transformers, numpy; print('torch:', torch.__version__); print('transformers:', transformers.__version__); print('numpy:', numpy.__version__); print('CUDA available:', torch.cuda.is_available())"
```
预期输出类似：
```
torch: 2.7.1
transformers: 4.56.0
numpy: 1.26.4
CUDA available: True
```

## 5. 可选：安装 tmux（用于后台运行训练）
```bash
apt-get update && apt-get install tmux -y   # Ubuntu/Debian
# 或使用 conda 安装：conda install -c conda-forge tmux
```

## 6. 可选：安装 jupyter/lab（如需交互式调试）
```bash
pip install jupyter jupyterlab
```

---

**环境部署完成**。  
接下来即可将训练脚本 `train_final.py` 和数据文件 `wechat_stream.jsonl` 放入工作目录，并按流程启动训练。
